## معرفی پروژه

این پروژه یک **ابزار مقایسه محصولات** با قابلیت انتخاب حداکثر ۴ آیتم از بین **۱۰۰+ محصول** است. کاربر می‌تواند محصولات را جستجو کرده، به لیست مقایسه اضافه کند و تفاوت‌های مشخصات فنی آن‌ها را به صورت بصری مشاهده نماید.

**هدف اصلی:** پیاده‌سازی با **استانداردهای عملکردی** و رعایت دقیق نکات ارزیابی شامل مدیریت بهینه DOM، الگوریتم‌های کارآمد، کنترل interactions و مدیریت حافظه.

---

## ویژگی‌ها

- **لیست پویا** با بیش از ۱۰۰ محصول
- **انتخاب حداکثر ۴ محصول** برای مقایسه
- **امکان جستجو** با قابلیت تایپ و جستجو با Enter
- **مقایسه ستونی** با نمایش تمام مشخصات فنی
- **هایلایت هوشمند** تفاوت‌ها بین محصولات

### UI/UX

- **دکمه شناور** با نمایش تعداد انتخاب‌ها
- **لودر سراسری** (Overlay) هنگام جستجو
- **نوتیفیکیشن** برای عملیات‌های مختلف
- **طراحی واکنش‌گرا** برای موبایل و تبلت
- **انیمیشن‌های روان** برای تعاملات کاربر

### Technical Features

- **مدیریت state** با localStorage
- **Event Delegation** برای تمام رویدادها
- **Debounce** برای بهینه‌سازی جستجو
- **الگوریتم مقایسه** با پیچیدگی O(n)
- **مدیریت حافظه** و پاکسازی خودکار

## ساختار پروژه

```
product-comparison/
│
├── index.html              # صفحه اصلی (داشبورد)
├── products.html           # صفحه لیست محصولات
├── compare.html            # صفحه مقایسه
├── style.css               # استایل مشترک
│
├── data/
│   └── mockData.js         # ۱۰۰ محصول با مشخصات کامل
│
├── utils/
│   ├── storage.js          # مدیریت state با localStorage
│   ├── comparison.js       # الگوریتم مقایسه
│   └── helpers.js          # توابع کمکی
│
└── page/
    ├── home.js             # منطق صفحه اصلی
    ├── products.js         # منطق صفحه محصولات
    └── compare.js          # منطق صفحه مقایسه
```

### اجرا با Live Server

live-server

یا مستقیماً فایل `index.html` را در مرورگر باز کنید.

## راهنمای استفاده

### صفحه اصلی (Home)

- مشاهده آمار کلی محصولات
- دسترسی سریع به صفحات محصولات و مقایسه

### صفحه محصولات (Products)

- **جستجو**: عبارت مورد نظر را تایپ کرده و Enter بزنید یا روی دکمه جستجو کلیک کنید
- **انتخاب محصول**: روی دکمه "➕ انتخاب" کلیک کنید (حداکثر ۴ محصول)
- **حذف از مقایسه**: روی دکمه قرمز "🗑️ حذف از مقایسه" کلیک کنید
- **دکمه شناور**: پس از انتخاب محصولات، دکمه شناور در پایین صفحه ظاهر می‌شود
- **مشاهده مقایسه**: روی دکمه شناور کلیک کنید

### صفحه مقایسه (Compare)

- مشاهده جدول مقایسه محصولات انتخاب شده
- **هایلایت زرد**: مشخصاتی که بین محصولات متفاوت هستند
- **حذف تکی**: روی دکمه قرمز بالای هر محصول کلیک کنید
- **پاک کردن همه**: روی دکمه "🗑️ پاک کردن همه" کلیک کنید

## بهینه‌سازی‌های انجام شده

### **مدیریت DOM**

- استفاده از `DocumentFragment` برای رندر گروهی
- به‌روزرسانی فقط دکمه‌ها (نه کل کارت) هنگام تغییر state
- کش کردن المنت‌ها برای دسترسی سریع‌تر

### **کنترل Interactions**

- **Debounce** برای جستجو (تاخیر ۱.۵ ثانیه)
- **Throttle** برای رویدادهای مکرر
- **Event Delegation** برای کاهش تعداد listenerها

### **مدیریت حافظه**

- پاکسازی تایمرها در `beforeunload`
- استفاده از `WeakRef` (در نسخه پیشرفته)
- عدم نگهداری رفرنس‌های اضافی

### **الگوریتم‌ها**

- مقایسه با پیچیدگی O(n)
- کش کردن تفاوت‌ها برای رندر مجدد
- استفاده از Set برای جستجوی سریع‌تر

### **State Management**

- ذخیره در localStorage برای ماندگاری
- استفاده از Set برای جلوگیری از تکرار
- انتشار رویداد برای هماهنگی بین صفحات
